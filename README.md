# Shafar Foundation API

A scalable backend system built with **Golang** to manage the digital infrastructure of the **Shafar Foundation**. This API handles everything from volunteer recruitment and project management to real-time donation tracking and financial auditing.

## Key Features
- **User & Role Management:** Secure registration and login for Volunteers, Committee Members, and Admins.
- **Project Tracking:** Create and manage foundation initiatives (e.g., Blood Donation camps, Food distribution).
- **Financial Module:** Comprehensive tracking of **Incomes**, **Expenses**, and **Donations**.
- **Automated Calculations:** Real-time updates of project funds when donations are approved.
- **Type-Safe API:** Built with Go's strict typing and Gin's high-performance routing.

## Tech Stack
- **Language:** Go (1.25+)
- **Web Framework:** [Gin Gonic](https://github.com/gin-gonic/gin)
- **Database (ORM):** [GORM](https://gorm.io/)
- **Database Engine:** PostgreSQL
- **Data Format:** JSON
